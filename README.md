# ЭКО КРСК - Экологические технологии в Красноярске

Официальный сайт компании ЭКО КРСК - ведущего поставщика экологических технологий и инженерных решений в Красноярске.

## 🚀 Деплой на Vercel

1. Подключите репозиторий к Vercel
2. Настройки деплоя:
   - **Framework Preset**: Other
   - **Build Command**: `echo 'Build complete'`
   - **Output Directory**: `.` (корневая папка)
   - **Install Command**: (оставить пустым)
